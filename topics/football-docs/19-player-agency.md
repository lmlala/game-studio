# AGY — 玩家代理感：可控面的设计与取舍

> 目标体验：**「因为我的选择，球队最后赢了；因为我当年顶住压力留下
> 老队长，他今年拿了年度最佳。」** 本文件回答：玩家到底控制什么、
> 每个控制的杠杆多大、系统如何让玩家*相信并看见*是自己造成的——
> 有限控制（G1）不等于无力感，这条线怎么画。

### AGY-01 代理感三支柱 [总纲]

状态: draft · 优先级: P0 · 依赖: 无

**目的**：给"玩家觉得是自己赢的"一个可工程化的定义，使其可设计、
可测量，而不是玄学。

**设计理念**：代理感 = 杠杆真实 × 后果可见 × 归因可信，三者缺一即
崩：杠杆假 → 安慰剂感；后果不可见 → 石沉大海感；归因不可信 →
骰子感。三支柱分别由 AGY-02 / AGY-03 / PSY-05 承载。

**如何设计**：
- 支柱一 **杠杆真实**：每个玩家动作的效果量必须显著大于系统噪声
  （机器可验：同 seed 反事实对比，AGY-02 的逐动作验收）；
- 支柱二 **后果可见**：每个重大决策在 1-3 周内必有可见回响
  （PSY-02 已定），且回响明确标记玩家决策为因（AGY-03 的玩家决策节点）；
- 支柱三 **归因可信**：负面结果有前兆链、正面结果有玩家贡献链，
  但归因呈现**克制**——系统展示因果，玩家自己得出"是我"
  （PSY-05 的"不打分不审判"纪律）；
- [锁死] 反支柱禁令：禁止虚假代理（选项不同文本后果相同）、禁止
  安慰剂控制（设置无效但展示"已生效"）、禁止抢功文案（系统替玩家
  宣布"你的英明决策…"——肉麻即出戏）。

**验收标准**：
- [机器] 全部玩家动作通过 AGY-02 的反事实效果量测试；
- [机器] 重大决策回响覆盖率 100%（PSY-02 复引）；
- [人工] 内测问卷「胜利/失败主要因为我的决策」认同率 ≥ 60%，
  且与实际决策质量正相关（赢家和输家都认同才算成立）。

**评估钩子**：
- 反事实效果量榜（AGY-02）；决策回响延迟分布。

### AGY-02 玩家动词清单与杠杆值

状态: draft · 优先级: P0 · 依赖: AGY-01, NUM-02

**目的**：穷举玩家全部可执行动作（动词），为每个动词定额杠杆——
取舍的核心工具：杠杆给谁、给多大、多久兑现，一张表说清。

**设计理念**：决策权重金字塔——少数高杠杆"重决策"制造紧张
（首发/带伤/处罚），大量中低杠杆"经营动作"制造掌控节奏（训练/
谈话）。每动词三属性：效果量级（大/中/小）、兑现期（即时/周/赛季）、
反噬面（哪些群体可能因此恨你）。

**如何设计**：
- 动词总表（杠杆量级 [锁死]，具体数值黄色参数走 NUM-06）：
  | 动词 | 杠杆 | 兑现期 | 反噬面 | 来源卡 |
  | --- | --- | --- | --- | --- |
  | 首发 11 人 | 大 | 即时 | 替补怨气 | UX-02 |
  | 带伤派遣 | 大 | 即时+赛季 | 球员/更衣室 | CAT-02 |
  | 战术五项 | 中 | 即时 | 执行度反噬 | MAT-02 |
  | 临场调整(≤4次) | 中 | 即时 | 被换者 | UX-04 |
  | 训练重点/强度 | 中 | 周-月 | 伤病/抱怨 | CAT-01 |
  | 单人关注(≤3) | 中 | 月 | 未被关注者 | CAT-01 |
  | 承诺 | 大(延迟) | 赛季 | 违诺即背叛 | ACT-04 |
  | 处罚/表扬 | 中 | 周 | 派系反弹 | ACT-03 |
  | 队长任命 | 大 | 赛季 | 落选派系 | REL-03 |
  | 发布会口径(3选1) | 小中 | 周 | 媒体扭曲 | CAT-03 |
  | 转会/续约建议 | 中(受老板裁决) | 窗口期 | 老板/球员 | CAT-04 |
  | 室友/分组安排 | 小 | 月 | 低感知慢杠杆 | REL-02 |
- [锁死] 杠杆下限：任何"大"杠杆动词的反事实效果量 ≥ 3× 系统噪声，
  "中" ≥ 1.5×，"小" ≥ 0.8×（可被察觉但需积累）；
- [锁死] 杠杆上限：没有动词能单独保证赢球——大杠杆改变概率分布
  ≤ ±15pp，决定论是代理感的另一种死法（全能感杀死戏剧，G1）；
- 反事实测试方法：同 seed 同周，动词取两个极值各跑 200 次，比较
  目标指标分布（KS 距离 → 效果量）——每动词一个标准测试，进 CI。

**验收标准**：
- [机器] 全动词反事实测试达到各自量级下限、不破上限；
- [机器] 动词表与 UX-02/各来源卡的操作清单一一对应（无表外动词、
  无无杠杆动词）；
- [人工] 任一动词被玩家社区标记"没用"时复测其效果量（上线后机制）。

**评估钩子**：
- 反事实效果量榜（每版本刷新）；动词使用率分布（某动词无人用 =
  杠杆或反馈设计失败）。

### AGY-03 玩家决策入因果链与"我的赛季"叙事

状态: draft · 优先级: P0 · 依赖: AGY-01, ENG-04, UX-06

**目的**：让玩家的决策在因果链与战报中**有名有姓**——支柱二的
机制落地。系统不缺因果（ENG-04），缺的是把"玩家这个人"在因果里
高亮出来。

**设计理念**：玩家决策是一等公民 cause：每个玩家命令生成
`player_decision` 节点进因果 DAG；下游任何事件都能回溯到它，
UI 与战报对这类节点做专属呈现。

**如何设计**：
- [锁死] `player_decision_id` 进日志 schema（ENG-04 扩展字段）：
  记录动词、参数、当时可见的信息快照（事后复盘"我当时知道什么"
  ——避免上帝视角悔棋感）；
- 因果板（UX-06）玩家节点专属样式："你的决定"徽标 + 当时情境
  一行（『第 9 轮，你顶住老板压力继续首发老队长』）；
- 战报新增固定栏目「主教练的赛季」（UX-05 扩展）：
  影响力 Top5 决策（影响力 = 下游因果子树的加权规模，severity
  加权）、最划算决策 / 最昂贵决策、未走之路提示（重大决策点的
  另一个选项 + 一句"无人知晓结果"——克制的反事实，不真跑模拟）；
- 长线印记：玩家对 Actor 的关键决策（提拔/保护/承诺兑现/牺牲）
  写入**玩家印记记忆**（ACT-03 的子类型，衰减减半）——这是
  AGY-04 长线回报的存储基础；
- 克制纪律：玩家节点只做视觉高亮与聚合统计，文案不评价不吹捧
  （AGY-01 反支柱禁令）。

**验收标准**：
- [机器] 全部玩家命令产生 player_decision 节点；任意 notable 事件
  的因果板可显示其玩家祖先节点（若存在）；
- [机器] 「主教练的赛季」栏目在跑批战报中 100% 生成且决策引用可
  回溯；
- [LLM] 评估器读战报后能正确复述「玩家本季三个最重要决策及其
  后果」≥ 4/5。

**评估钩子**：
- 因果板中玩家节点的出现率（健康带：notable 事件的 30-60% 能
  回溯到玩家决策——过低=世界与玩家无关，过高=世界没有自主性）。

### AGY-04 长线代理：「老队长拿年度最佳」回路

状态: draft · 优先级: P0 · 依赖: AGY-03, ACT-04, DIR-05

**目的**：设计跨赛季的"养成-荣耀"回路：玩家多年的培养、保护、
信任，最终凝结成 NPC 的高光时刻并**明确回敬玩家**——你点名的
体验原型的机制化。

**设计理念**：长线代理 = 印记积累 + 荣耀时刻 + 回敬呈现。关键是
回敬必须由系统**自然长出**（颁奖词引用真实记忆），不是剧本桥段。

**如何设计**：
- 印记账本：每 Actor 维护对玩家的印记记忆集（AGY-03 子类型）：
  被提拔首秀 / 被保护过 / 承诺被兑现 / 低谷被信任 / 被牺牲（负面
  印记同样入账——背叛的长线回敬是复仇，DIR-05 恩怨账本联动）；
- 荣耀时刻类型（赛季结算事件池）：年度最佳 / 金靴 / 最佳新人 /
  百场里程碑 / 告别战 / 入选名人堂（多赛季后）——荣耀判定纯看
  表现数据，**不为玩家偏置**（可信度优先）；
- 回敬机制：荣耀时刻触发时扫描得主的印记账本，显著度 top 印记
  生成回敬内容——获奖感言引用（『他说他不会忘记第 2 个赛季
  所有人都要卖掉他的时候，是谁把队长袖标给了他』）、媒体回顾版
  （CAT-03）、玩家声望与老板满意度结算；
- 失败侧同构：被你牺牲的球员在别队夺冠 / 复仇进球后，同一回敬
  管线输出苦涩版本（G7：失败叙事同样优先）；
- 节奏控制：回敬是稀缺品——每赛季玩家收到的强回敬 1-3 次
  （NUM-05 配额表新增行），多则廉价。

**验收标准**：
- [机器] 20 seed 跑批（玩家策略模拟为"养成倾向"脚本）：≥ 30% 的
  赛季个人奖项得主持有玩家印记，其中 ≥ 80% 触发回敬内容；
- [机器] 荣耀判定与玩家身份无关（同表现不同归属，获奖率无差异
  ——NUM-07 橡皮筋禁令的延伸验证）；
- [LLM] 回敬文本盲评「真挚不肉麻」≥ 4/5；
- [人工] 内测访谈出现自发复述"我培养的 XX 后来…"故事 ≥ 50%。

**评估钩子**：
- 印记记忆的写入/引用率；回敬事件的玩家停留时长（UI 埋点）。

### AGY-05 可控面取舍原则 [锁死]

状态: draft · 优先级: P0 · 依赖: AGY-01, AGY-02

**目的**：给"什么该给玩家控制"一个可执行的判定程序——未来每次
"要不要加个按钮"的争论都用本卡裁决，防止控制面无限蔓延成 FM。

**设计理念**：控制面是体验的负空间：每多给一分控制，不确定性
就少一分戏剧。取舍标准不是"玩家想不想要"，而是"失去这分不确定
会不会杀死故事"。

**如何设计**：
- [锁死] 四问判定程序（全过才能进控制面）：
  1. **职业边界**：现实主教练能做吗？（不能 → 拒绝，G1）；
  2. **戏剧保护**：控制它会消灭哪类事件？消灭的事件池占比 > 2%
     → 拒绝（如：可指定罚点球者会消灭"抢点球"冲突）；
  3. **决策密度**：每周决策数会超 3 吗？（UX-02 [锁死]）超 →
     必须同时降级一个现有动词为默认自动；
  4. **杠杆诚实**：能给出 ≥ 0.8× 噪声的真实效果吗？不能 → 拒绝
     （宁缺毋假，AGY-01 反支柱）；
- [锁死] 永不下放清单（G1 的细化）：球员临场具体跑位与传球选择 /
  球员内心状态直改 / 记者标题 / 老板决策 / 球迷情绪 / 派系结盟 /
  其他俱乐部内政——这些是世界自主性的核心资产；
- 给方向不给执行 [锁死]：所有控制都是"意图表达"，执行经过人格
  与关系的折射（训练设重点不设动作、转会提建议不签字、战术给
  倾向球员有执行度）——折射层就是故事发生的地方；
- 控制面变更流程：新增/移除动词必须更新 AGY-02 总表 + 跑反事实
  测试 + 过四问记录（写入卡片修订理由）。

**验收标准**：
- [机器] 控制面与 AGY-02 表严格同步（代码命令枚举 ↔ 表行数一致）；
- [人工] 每个新增动词的四问记录存在且结论可复核；
- [机器] 每周强制决策数 ≤ 3 恒成立（UX-02 复引，跑批断言）。

**评估钩子**：
- 动词增减历史曲线（持续膨胀 = 取舍纪律失效的预警）。

### AGY-06 失败的代理感

状态: draft · 优先级: P1 · 依赖: AGY-01, PSY-04, PSY-05

**目的**：保证失败同样被体验为"我的选择的后果"而不是"系统害我"
——代理感在输的时候比赢的时候更难也更重要（输家留存的根）。

**设计理念**：悲剧代理 =「我当时只能这么做，但后果真的追上来了」
（玩法文档核心爽点的机制化）：失败必须可归因到一连串各自合理的
玩家决策，而非单点失误或纯厄运。

**如何设计**：
- 失败复盘页（PSY-04 连败复盘的代理感版）：按时间线列出玩家
  决策节点 + 当时信息快照 + 实际后果——重点呈现「以当时所知，
  这个选择并不蠢」（信息快照是关键：AGY-03 已存）；
- 两难存档：Director 保证赛季内玩家经历 ≥ 2 次真两难（所有选项
  都有代价的决策卡，EVT 池标记 dilemma 标签）——失败若源于两难，
  玩家归因于命运与取舍而非自己愚蠢（健康的失败归因）；
- 厄运与决策分账：复盘页区分「决策链后果」与「纯厄运事件」
  （NUM-03 判定噪声所致），厄运部分明确标注——系统主动认领
  随机性，换取玩家对决策部分归因的信任；
- 禁止马后炮：复盘不展示"最优解对比"（没有最优解是设计事实，
  G1），只展示"未走之路无人知晓"（AGY-03 同款克制）。

**验收标准**：
- [机器] 失败赛季（降级/下课档）的复盘页 100% 可生成，决策节点
  ≥ 5 个且信息快照完整；
- [机器] dilemma 标签事件每赛季 ≥ 2 次触达玩家（跑批断言）；
- [人工] 内测失败玩家访谈：「输是因为系统耍赖」选择率 < 15%，
  「想再开一档证明自己」≥ 50%（PSY-04 继续游玩率的归因侧验证）。

**评估钩子**：
- 失败档的复盘页打开率与后续行为（再开档/弃游）的相关性。
