# 游戏总览（agent 常驻上下文文档）

> **用途**：设计 agent 处理任何卡片时的第一份上下文（配方见文末）。
> **纪律**：本文是全库唯一的"游戏是什么"权威摘要，预算 ≤ 2000 token；
> 内容变更视同改宪法，仅人工修改并签收，agent 只读。

## 一句话

**my-ft**：Steam 独立足球叙事经理。玩家是架空足球世界的新手主教练，
用**有限控制**影响一支混乱球队，看人心、媒体、老板与记忆交织成赛季
故事——"RimWorld 式的足球更衣室故事生成器"。

## 商业定位

- 切口：FM 玩家中的叙事疲劳者 × RimWorld/CK 系统叙事玩家的交集；
  当前 FM26 口碑崩盘是品类窗口期；无直接竞品。
- 卖点三件套：会记住你的球员 / 会失控的更衣室 / 会讲故事的赛季。
- 形态：买断制单机、中文优先后出英文、个人副业产能（窄而深）。
- 不做：FIFA 式操作、FM 数据库克隆、真实授权、无后果弹窗合集。

## 核心幻想与玩家可控面

幻想：「信息不完整、人心不稳定、外压不断变化下，靠判断、妥协和冒险
把球队带向某种结局」——结局可以是冠军，也可以是一段荒诞失败史。

可控：阵容/阵型/心态节奏压迫/核心角色/训练/谈话/处罚/承诺/转会建议。
不可控：球员真心、老板意图、记者标题、球迷情绪、派系串联、临场心态。

## 核心循环（周）

复盘上周事件 → 处理 0-3 个决策卡 → 训练设置 → 首发与战术 →
比赛（文本直播/跳过）→ 赛后发布会与更衣室余波 → 关系/记忆/联赛更新。

## 世界观要点

- **足球规则不架空**（11 人制、积分、红黄牌、转会窗全部现实可信）；
  **足球世界架空**（虚构小国瓦洛尼亚联盟：经济停滞、足球是社会情绪
  出口；5 个地区性格、俱乐部文化标签、老板/媒体/球迷/协会各有利益）。
- 基调：黑色幽默来自高压职业体育的利益错位；荒诞必须有因果锚点
  （性格/利益/压力/误会/记忆），分级 0-2，禁止超现实。

## 目标用户（四画像一行版）

1. FM 叙事疲劳者：要球员像人、事件不模板化；
2. 系统叙事玩家（核心）：要派系涌现、失败叙事、因果可查；
3. 轻度足球玩家：要每周 ≤3 个决策、情绪过山车；
4. 内容创作者：要战报、seed 名片、名场面可分享。
冲突仲裁优先序：2 > 1 > 4 > 3。

## 品味锚点（成功句式样例）

- 「为保级把带伤队长派上场，他进了关键球但赛季报销；半年后年轻球员
  开始觉得我会牺牲球员。」
- 「替补门将因决赛被临时拿下而记恨我，后来加入老人帮，连续在媒体前
  暗示更衣室不团结。」
  ——任何设计若不能服务于产出这类「可复述的因果故事」，需质疑其优先级。

## 范围红线（设计时随时核对）

- MVP：1 国 1 联赛 8-12 队、每队 18-22 人、双循环、文本比赛、
  事件池 80-120、单赛季闭环 + 存读档 + 战报导出；
- 个人副业产能（G8）：任何卡片先问"一个人业余时间能否做完最小版"；
- 技术：Rust 权威模拟（确定性 seed、结构化日志、CLI 无头跑批），
  Godot 仅展示；现有代码可完全重构对齐卡片。
